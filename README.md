# nodejs-web-api-express-params-hello-world

## Description
A basic web api that takes parameters.

## Tech stack
- nodejs

## Docker stack
- node:latest

## To run
`sudo ./install.sh -u`
- http://localhost
- http://localhost/<name>
  - <name> being anything

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
